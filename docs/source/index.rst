Welcome to Krajjat's documentation!
===================================

**Krajjat** (originally, Kinect Realignment Algorithm for Joint Jumps and Twitches)
is a Python toolbox for processing MoCap data (from Kinect v2 or Qualisys systems) and
find relationships between the tracked movements and an audio file.
