Time Series
===========

Description
-----------
This class is the parent class for :class:`Sequence`, and all of the classes
derived from :class:`AudioDerivative`: :class:`Audio`, :class:`Envelope`,
:class:`Pitch`, :class:`Intensity`, and :class:`Formant`.
